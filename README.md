# VitBer TekFys Project

This project created during the course "TMA4320 Introduction to Scientific Computation" at NTNU. The assignment description and the assignment solution can be found in the folder 'Main'. The topic of this assignment is within the field of engeneering physics (Norwegian teknisk fysikk), hence 'TekFys' in the title.
